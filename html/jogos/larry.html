Grow Up Larry

Engine: Unity
Grow Up Larry é um jogo 2D no estilo Endless Runner (estilo de jogo onde o jogador precisa percorrer uma trajetória infinita acumulando o máximo de pontos possíveis, alguns exemplos conhecidos são Subway Surfers e Jetpack Joyrider). O jogador controla Larry, um simpatico cubo que é capaz de alterar seu tamanho e quando maior fica, mais pontos por segundo ele acumula, o jogo foi desenvolvido durante uma game jam onde o tema era "Scale" (podendo ser traduzido como Escama ou Escala)
Esse jogo foi criado durante a participação da game jam: Game Off 2023
Link da Game Jam: https://itch.io/jam/game-off-2023
