Grow Up Larry

Engine: Unity
Grow Up Larry is a 2D Endless Runner game (a game style where the player needs to traverse an infinite path while accumulating as many points as possible; some well-known examples include Subway Surfers and Jetpack Joyride). The player controls Larry, a friendly cube that can change its size, and the larger it gets, the more points per second it accumulates. The game was developed during a game jam where the theme was 'Scale'.
This game was created during the Game Off 2023 game jam.
Game Jam link: https://itch.io/jam/game-off-2023
